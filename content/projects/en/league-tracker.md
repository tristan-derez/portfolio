### Description

League Tracker is a Discord application that you can add to a text channel in your discord server.

Add summoners with their summoner's name:
```bash
/add summonersName#EUW
```
As soon as they complete a game, the app will send a message in the text channel:

![discord screenshot](https://i.ibb.co/KxzMYkWX/Screenshot-2026-05-25-231200.jpg)

---

### Features:

- Real-time ranked game tracking
- Rank change notifications
- Detailed per-game statistics
